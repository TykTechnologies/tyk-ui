**Panel**

*Panel themes*
```js
<Panel theme="primary">
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
<Panel theme="success">
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
<Panel theme="danger">
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
<Panel theme="warning">
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
```
*Collapsable panel*
```js
<Panel theme="primary" collapsable={ true }>
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
```
```js
<Panel theme="primary" collapsable={ true } collapsed={ true }>
  <Panel.Header>Panel example</Panel.Header>
  <Panel.Body>
    Panel body content
  </Panel.Body>
</Panel>
```
